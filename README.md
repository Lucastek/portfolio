# Sales Performance Analysis & Business Insights

## Project Overview
This project aims to analyze commercial performance data in order to identify key business drivers, detect performance issues, and provide actionable insights for decision-makers.

The analysis focuses on sales, customers, products, and time-based trends to support strategic and operational decisions.

## Business Problem
The sales department has access to multiple data sources but lacks clear visibility on:
- key performance drivers
- high-value customer and product segments
- underperforming regions or periods

The objective is to leverage data analytics to improve business monitoring and support decision-making.

## Tech Stack
- Python (pandas, NumPy, Matplotlib)
- SQL (SQLite / PostgreSQL)
- Power BI
- Git & GitHub
- Jupyter Notebook

## Project Structure
- `data/`: raw and processed datasets
- `notebooks/`: data exploration, cleaning, and analysis
- `sql/`: analytical SQL queries
- `powerbi/`: interactive dashboards
- `visuals/`: exported charts and visuals

## Key Deliverables
- Cleaned and structured datasets
- Exploratory and analytical notebooks
- SQL analysis queries
- Interactive Power BI dashboard
- Business-oriented insights and recommendations
